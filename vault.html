Password Vault by Benjamin Miller

Stored Credentials

How to Use:

  1. Enter the website name in the first input field.
  2. Type the username you wish to use.
  3. Input the password and hit "Save Credentials".
  4. To load and view saved credentials, hit the "Load Credentials" button.
  5. To upload an encrypted file with credentials, click "Upload File".
